Files src/io/fileIO.hpp
src/io/fileIO_model.hpp
src/io/fileIO_dat.hpp

Description

UnicornTK relies on the extension of the input file to determine its type. When the extension is .dat, the pointsets are stored in ASCII mode, without any header. It separates pointsets using "#".
