README

Cloning the repository

git clone https://github.com/utk-team/utk.git

Compiling the code 

cd utk/src
mkdir build
cd build
cmake ..
make

By default, no external library is needed to compile UTK. Tools that require extrernal libraries are therefore not compiled. To compile such tools, one can use the ccmake tool, to activate the required external libraries, before running cmake and make again. For example, to run the spectral test, one needs to activate the FFTW and CIMG libraries.

Using the code

The UTK library contains a series of executables to evaluate and generate point sets, organized as follows (from the build folder):

bin/
-------- discrepancy/
------------------------ BoundariesStarDiscrepancy_fromfile_*d* (Code from [Thiemard01] )
 ------------------------ CenteredL2Discrepancy_fromfile_*d* (computes the exact centered l_2 star discrepancy of an input point set [Hickernell98])
------------------------ GeneralizedL2Discrepancy_fromfile_*d* (computes the exact generalized l_2 star discrepancy of an input point set [Hickernell98])
------------------------ L2Discrepancy_fromfile_*d* (computes the exact l_2 star discrepancy of a point set)
------------------------ StarDiscrepancy_fromfile_*d* (computes the exact l_infinite star discrepancy of a point set)
------------------------ SymmetricL2Discrepancy_fromfile_*d* (computes the exact symmetric l_2 star discrepancy of an input point set [Hickernell98])
------------------------ UnanchoredL2Discrepancy_fromfile_*d* (computes the exact unanchored l_2 star discrepancy of an input point set [Hickernell98])
-------- integration/
------------------------ IntegrationCenteredGaussian_fromfile_*d* (integrates a 2D gaussian with mean at 0.5,0.5 with an input point set)
------------------------ IntegrationCorneredGaussian_fromfile_*d* (integrates a 2D gaussian with mean at 0,0 with an input point set)
------------------------ IntegrationGaussian_fromfile_*d* (integrates a parametrable 2D gaussian (but without analytic value) with an input point set)
------------------------ IntegrationNSphere_fromfile_*d* (integrates a N-dimensional spjere with an input point set)
------------------------ IntegrationImage2D_fromfile_*d* (integrates a 2D image (.exr ideally) with an input point set) [CIMG Required]
------------------------ IntegrationBoxArrangement_fromfile_*d* (integrates an analytical box arrangement with an input point set) [CGAL Required]
------------------------ IntegrationTriangleArrangement_fromfile_*d* (integrates an analytical triangle arrangement with an input point set) [CGAL Required]
-------- samplers/
------------------------ AAPatterns_2dd ([xxxxxxxx])
------------------------ CMJ_2dd ([Kensler03])
------------------------ DartThrowing_2dd (naive)
------------------------ FastPoisson_2dd ([xxxxxxxx])
------------------------ Faure_2d* ([xxxxxxxx])
------------------------ Forced Random_2dd ([xxxxxxxx])
------------------------ Halton_*dd ([xxxxxxxx])
------------------------ Hammersley_2dd ([xxxxxxxx])
------------------------ LDBN_2d* ([xxxxxxxx])
------------------------ Nieddereiter_*d* ([xxxxxxxx])
------------------------ NRooks_*dd ([xxxxxxxx])
------------------------ Owen_*dd ([xxxxxxxx])
------------------------ Penrose_2dd ([xxxxxxxx])
------------------------ Rank1_2dd ([xxxxxxxx])
------------------------ RegularGrid_*dd
------------------------ Sobol_2dd ([xxxxxxxx])
------------------------ SobolIndices_*d* (Polynomials as parameters, indices from [xxxxxxxx])
------------------------ Stratified_*dd
------------------------ Whitenoise_*dd
-------- spectral/
------------------------ stkfourier_2dd (Discrete Fourier Trabsform, only 2D) [FFTW and CIMG Required]
-------- statistics/
------------------------ computeMinDist_2dd (returns the minimal distance between pairs of samples)
------------------------ computePCF_2dd (computes the PCF of a 2D point set [Oztirelli])
-------- render/
------------------------ zoneplate_fromfile_2dd (reconstructs a zoneplate using a Mitchell filter and samples from an input point set) [CIMG Required]
-------- utils/
------------------------ draw_fromfile_2dd (draws a 2D point set to a .ps/.tikz file)

Note that this list is not exaustive, the library may contain other tools, that are not listed here and they are still in an early version.

To get the list of parameters for each executable, one can use the -h option. For example with the whitenoise sampler 

./bin/samplers/Whitenoise_2dd -h
