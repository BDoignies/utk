The Pointset Class

This class is at the core of all the UTK framework. Each tool manipulates this structure to generate or analyze point sets. Since the main objectiv of this framework is to ensure as much modularity as possible, this class is very simple.

This class inherits from the std::vector class, and is templated using 3 parameters:

  • The number of dimensions of the samples
  • The type of the coordinated (usually int,float or double)
  • The type of point (utk::Point / utk::TaggedPoint). The templated values of the Point used should match the first template values of the Pointset class.
For example, to instanciate a 2D pointset with floating point value, one will use the following C++ code: 
utk::Pointset< 2, double, utk::Point< 2, double > >
However, the following code will generate an error: 
utk::Pointset< 2, double, utk::Point< 2, int > >
