The Vector Class

This class represents a D-dimensional vector, with several operators.

The Point Class

This class represents a sample. It its current state, it only contains a utk::Vector that represents the position of the sample.

The TaggedPoint Class

This class inherits from the Point class and adds a templated-type value that can be used as a tag for the sample.

The Domain Class

This class contains two samples, the minimal and maximal corners of a D-dimensional interval.
